üí° PowerPoint Tips

  1. Never use PowerPoint's built-in chart title ‚Äî it's small and badly positioned.
  2. Instead, add a Text Box ABOVE the chart.
  3. Title: 20‚Äì24pt, Bold, dark color.
  4. Subtitle: 12‚Äì14pt, Regular, grey color.
  5. Position: Left-aligned with the chart's left edge.
  6. Use Format > Text Box > Internal margins to fine-tune alignment.
  7. Template tip: Create a "Chart Slide" layout in Slide Master with pre-positioned title/subtitle text boxes.
